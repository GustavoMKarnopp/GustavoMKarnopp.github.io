SOBRE MIM

Desenvolvedor Front end, estudante, amante da tecnologia, buscando aprender e disposto a sempre dedicar o melhor de mim em todos os locais e projetos que atuo. Conhecimento e experiência em linguagem Javascript, frameworks como Vue.js, Vuetify, HTML, CSS. Experiência com ferramentas de versionamento de código como git e bitbucket. Participação em diversos projetos, tanto profissionais quanto pessoais. Natural do município de Vera Cruz - Rio Grande do Sul/ Brasil, sempre tive interesse por inovação tecnológica mas nunca imaginei ingressar na área da programação. O tempo foi passando e entrei para o Exercito brasileiro, onde tive a oportunidade de aprender muita coisa e conhecer muitas pessoas, uma delas meu amigo e cumpadre, FELIPE CARPES, que hoje atua como desenvolvedor Front-end também, mas na época estava iniciando na peleia (Giria usada para batalha aqui no Sul) da vida de DEV rsrsrs. Em 2019 já cabo engajado do EB, vi a necessidade de escolher algo para meu futuro, ou seja uma profissão, pois meu tempo dentro do exercito já estava se esgotando, vindo então em mente as conversar e trocas de experiencias sobre a área da programação. Pesquisei mais sobre, executei alguns cursos do Gustavo Guanabara, e segui a diante nos estudos para ter a area de dev com minha profissão. Em 2021 surgiu a primeira oportunidade de eu por em pratica tudo oque já havia aprendido, com uma vaga de Trainee em uma empresa aqui da região da area de Boliche e Tickets/ Acessos - IMPLY, onde tive a oportunidade de aprender muito e alavancar minha carreira e experiencias com Desenvolvedor, ficando até o final de 2022, onde passei para uma nova fase da minha vida, fase esssa que hoje se chama Ernest Yang - EY... Obrigado se você leu até, esse foi um breve resumo de minha trajetória como desenvolvedor e profissional de TI.

O QUE EU FAÇO

Tipos de serviços e oque utilizo em meus projetos

VUE.JS Vue é um framework progressivo para a construção de interfaces de usuário. A biblioteca principal é focada exclusivamente na camada visual (view layer), sendo fácil adotar e integrar com outras bibliotecas ou projetos existentes.
HTML Linguagem de Marcação de Hipertexto". Consiste em uma linguagem de marcação utilizada para produção de páginas na web, que permite a criação de documentos que podem ser lidos em praticamente qualquer tipo de computador e transmitidos pela internet.
CSS3 Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página.
JAVASCRIPT é uma linguagem de programação que permite a você implementar itens complexos em páginas web

VUETIFY é uma biblioteca de interface do usuário sem habilidades de design necessárias com componentes Vue lindamente criados à mão.
UIKIT é um framework responsivo modular de front-end projetado para ser leve e simples de se usar.
TAILWIND é um framework CSS (Cascading Style Sheets) com classes predefinidas que você pode usar para construir e projetar páginas web diretamente na sua marcação. Ele permite que você escreva CSS em seu HTML na forma de classes pré-definidas.

ADOBE PHOTOSHOP é um software definido como editor de imagens. Desenvolvido pela Adobe Systems, o aplicativo, sem dúvidas, é o que possui maior destaque no mundo da fotografia e do design gráfico.
INKSCAPE é programa (software) de gráficos vectoriais de qualidade profissional que corre no Windows, Mac OS X e Linux. É utilizado por profissionais e amadores em todo o mundo, para criar um alargado leque de gráficos como ilustrações, ícones, logótipos, diagramas, mapas e gráficos para páginas da internet.

GIT é um sistema de controle de versão distribuído, o que significa que um clone local do projeto é um repositório de controle de versão completo.
BITBUCKET é mais do que apenas gerenciamento de código Git. Ele oferece às equipes um lugar onde planejar projetos, colaborar em códigos, testá-los e implementá-los.
